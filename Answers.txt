1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
  - проверить наличие в конфигурации git username + email (команда git config --global -l)
  - проверить, добавлен ли SSH ключ в профиле github.com
  - скопировать SSH ключ репозитория
  - в терминале перейти в директорию, в которую должна быть скопирована папка репозитория
  - в терминале запустить команду git clone <SSH ключ>
  - запустить команду git remote -v чтобы убедиться в источнике репозитория

2. Как сделать коммит?
  - git status - проверить список файлов, которые нужно подготовить для коммита
  - git add <file_name> - для добавления файлов к коммит
  - git commit -m "Relevant Comment" - для обновления файлов в локальном репозиторие
  - git push - для добавления изменений в удалённый репозиторий GitHub

3. Как получить на локальный компьютер изменения с github.com
  - запустить команду git pull - все изменения загрузятся

4. Какую информацию нам дают команда:
  - git status - показывает состояние файлов в локальной копии репозитория, предупреждает о файлах, не добавленных в commit
  - git log - показывает список всех коммитов, сделанных в репозитории, а также их время, автора и хеш-сумма
  - git remote -v - показывает ссылку на источник репозитория.
